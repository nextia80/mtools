# 좌측 메뉴 접기 및 단축키 정리

## 개요

좌측 사이드바를 접었다 펼칠 수 있는 기능을 추가했다.  
접힌 상태에서는 메뉴 번호 아이콘(1~7)만 보이고, 터미널 명령·하단 버튼·키보드 단축키로 제어할 수 있다.

---

## 1. 동작 방식

### 펼친 상태 (기본)

- 브랜드, 메뉴 이름, MD 하위 목록, 하단 안내 문구 모두 표시
- 사이드바 너비: `320px`

### 접힌 상태

- `1`~`7` 번호 아이콘만 표시 (호버 시 메뉴명 툴팁)
- MD 하위 목록, 새로고침 버튼, 하단 안내 문구 숨김
- 사이드바 너비: `4.5rem`
- 선택된 메뉴는 아이콘 색상으로 표시 (녹색)

---

## 2. 제어 방법

| 방법 | 접기 | 펼치기 |
|------|------|--------|
| 하단 버튼 | `<<` 클릭 | `>>` 클릭 |
| 터미널 명령 | `<` | `>` |
| 키보드 단축키 | `⌘ + ←` | `⌘ + →` |

### 터미널 출력

```text
> <
좌측 메뉴를 접었습니다.

> >
좌측 메뉴를 펼쳤습니다.
```

---

## 3. 변경 파일

| 파일 | 변경 내용 |
|------|-----------|
| `App.vue` | `sidebarCollapsed` 상태, 단축키, 터미널 액션 처리 |
| `SidebarMenu.vue` | `collapsed` prop, 하단 `<<` / `>>` 토글 버튼 |
| `app.css` | `.sidebar.collapsed`, `.app-shell.sidebar-collapsed` 스타일 |
| `terminal/index.ts` | `<`, `>` 명령 처리 |
| `terminal/types.ts` | `sidebar-set-collapsed` 액션 타입 추가 |

---

## 4. 구현 구조

```
사용자 입력
    ├── 하단 << / >> 버튼  → toggleSidebarCollapsed()
    ├── 터미널 < / >       → TerminalAction → sidebarCollapsed
    └── ⌘← / ⌘→           → handleKeyboardShortcut()
                ↓
         sidebarCollapsed (App.vue)
                ↓
    app-shell.sidebar-collapsed + SidebarMenu.collapsed
```

### TerminalAction 타입

```typescript
| { type: 'sidebar-set-collapsed'; collapsed: boolean }
```

터미널에서 `<` / `>` 입력 시 `executeTerminalCommand`가 위 액션을 반환하고,  
`App.vue`의 `handleTerminalAction`에서 `sidebarCollapsed` 값을 갱신한다.

## 5. CSS 요약

```css
/* 펼침 */
.app-shell { grid-template-columns: 320px minmax(0, 1fr); }

/* 접힘 */
.app-shell.sidebar-collapsed { grid-template-columns: 4.5rem minmax(0, 1fr); }

.sidebar.collapsed .brand-text,
.sidebar.collapsed .primary-menu-item strong,
.sidebar.collapsed .nested-md-menu { display: none; }
```

---

## 6. 기존 단축키와의 관계

| 단축키 | 동작 |
|--------|------|
| `⌘1` | 터미널 |
| `⌘2` | 게시판 |
| `⌘3` | 회원관리 |
| `⌘4` | 일정관리 |
| `⌘5` | API 스웨거 |
| `⌘6` | API 테스트 |
| `⌘7` | 일자별LOG |
| `⌘←` | **좌측 메뉴 접기** (신규) |
| `⌘→` | **좌측 메뉴 펼치기** (신규) |

메뉴가 접힌 상태에서도 `⌘1`~`⌘7`로 화면 전환은 그대로 동작한다.
