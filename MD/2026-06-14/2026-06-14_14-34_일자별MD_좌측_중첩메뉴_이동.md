# 2026-06-14 14:34 - 일자별 MD 좌측 중첩 메뉴 이동

## 질문 1. 어떤 요청이 있었나?

`일자별 MD`를 오른쪽 화면 상단 리스트가 아니라 좌측 메뉴에 붙이고 싶다는 요청이 있었다.

원하는 형태는 아래와 같았다.

```text
+ 홈
+ API 테스트
+ 일자별 MD
    + 20260614
      ..관련 파일
    + 20260613
      ..관련 파일
```

## 질문 2. 어떻게 변경했나?

좌측 메뉴의 `일자별 MD` 아래에 날짜별 그룹과 파일 목록을 중첩해서 표시하도록 변경했다.

변경 후 구조는 아래와 같다.

```text
좌측 메뉴
├─ 홈
├─ API 테스트
└─ 일자별 MD
   ├─ 20260614
   │  ├─ 2026-06-14_14-30_좌측메뉴_홈_API_일자별MD_분리.md
   │  └─ 2026-06-14_14-21_UI_좌측메뉴_레이아웃_정리.md
   └─ 20260613
      └─ 이전 작업 MD 파일들
```

날짜 그룹은 접고 펼칠 수 있는 `details / summary` 구조로 만들었다.

## 질문 3. 오른쪽 화면은 어떻게 바뀌었나?

기존에는 `일자별 MD` 화면 오른쪽 상단에 날짜별 MD 리스트가 보였다.

이번 변경 후에는 파일 목록을 좌측 메뉴로 옮겼기 때문에 오른쪽 화면에는 Markdown 뷰어 중심으로 표시된다.

```text
오른쪽 일자별 MD 화면
├─ 제목 영역
└─ 선택한 Markdown 내용
```

파일을 선택하지 않은 경우에는 아래 안내가 나온다.

```text
좌측 일자별 MD 메뉴에서 파일을 선택하세요.
```

```ts

```

```ts
const formatDateLabel = (date: string) => date.replace(/-/g, '')
```
