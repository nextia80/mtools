# 터미널 명령어 모듈 리팩토링

## 개요

`terminalCommands.ts` 단일 파일(약 770줄)에 몰려 있던 터미널 로직을 **역할별 모듈**로 분리했다.  
기능 변경 없이 구조만 정리했으며, type-check 통과를 확인했다.

---

## 변경 전 문제

| 문제 | 내용 |
|------|------|
| 단일 거대 파일 | `terminalCommands.ts` — bd / gc / md / API / 날짜 / 도움말 혼재 |
| 중복 가능성 | HTTP 호출·날짜 파싱·LIKE 검색이 명령마다 흩어질 위험 |
| 확장 어려움 | gc 명령 추가 시 700줄 파일 하단에 계속 누적 |

---

## 변경 후 폴더 구조

```text
mtools-ui/src/terminal/
├── index.ts                 # 진입점 — executeTerminalCommand, welcome
├── types.ts                 # TerminalAction, TerminalCommandResult
├── apiClient.ts             # fetchJson, requestJson, parseHttpError
├── dateUtils.ts             # parseYmd, buildDayRange, buildDateRange, getTodayYmd
├── textUtils.ts             # matchLike, extractKeyword, formatDateTime
├── help.ts                  # TERMINAL_HELP Map, /? 처리
└── commands/
    ├── boardCommand.ts      # bd 명령
    ├── calendarCommand.ts   # gc 명령
    └── mdCommand.ts         # md 명령
```

### 삭제된 파일

- `src/terminalCommands.ts`
- `src/terminalHelp.ts`

## 모듈별 역할

### `index.ts` — 라우터

- `executeTerminalCommand()` — `/?`, `cls`, `bd`, `gc`, `md` 분기
- `terminalWelcomeLines()` — 환영 메시지
- `TerminalAction`, `TerminalCommandResult` re-export

### `apiClient.ts` — HTTP 공통

- `fetchJson()` — GET JSON
- `requestJson()` — POST/DELETE 등
- `parseHttpError()` — Spring/Google API 에러 → 한글 메시지

### `dateUtils.ts` — 날짜 공통

- `parseYmd()` — `YYYYMMDD` 검증
- `buildDayRange()` — 하루 범위 (ISO timeMin/timeMax)
- `buildDateRange()` — 기간 범위
- `getTodayYmd()` — 오늘 날짜 문자열

gc 명령(`gc l`, `gc lc`, `gc lcb` 등)에서 공통 사용.

### `textUtils.ts` — 텍스트 공통

- `matchLike()` — `*` 와일드카드 LIKE 검색 (md 명령)
- `extractKeyword()` — 따옴표 제거
- `formatDateTime()` — 게시판 날짜 포맷

### `help.ts` — 도움말

- `TERMINAL_HELP` Map — `/?`, `/? gc` 등 데이터
- 새 명령 추가 시 **이 파일 Map만 수정**하면 도움말 자동 반영

### `commands/boardCommand.ts`

- `executeBdCommand()` — `bd l`, `bd g`, `bd d` 등

### `commands/calendarCommand.ts`

- `executeGcCommand()` — 조회(`gc l`, `gc c`, `gc lc` …) + 추가(`gc a`, `gc ad`, `gc at`)
- 캘린더 포맷·API 호출 포함

### `commands/mdCommand.ts`

- `executeMdCommand()` — `md ld`, `md l`, `md g`

## import 경로 변경

| 파일 | 변경 |
|------|------|
| `TerminalView.vue` | `'../terminal'` |
| `App.vue` | `'./terminal'` |

외부에서는 `terminal/index.ts`만 import하면 된다.

```typescript
import { executeTerminalCommand, terminalWelcomeLines } from '../terminal'
import type { TerminalAction } from '../terminal'
```

---

## 새 명령 추가 가이드

| 작업 | 수정 위치 |
|------|-----------|
| 도움말 | `terminal/help.ts` → `TERMINAL_HELP` |
| bd 하위 명령 | `terminal/commands/boardCommand.ts` |
| gc 하위 명령 | `terminal/commands/calendarCommand.ts` |
| md 하위 명령 | `terminal/commands/mdCommand.ts` |
| 새 명령 그룹 (예: `api`) | `commands/apiCommand.ts` + `index.ts` 분기 추가 |
| HTTP 공통 | `terminal/apiClient.ts` |
| 날짜 공통 | `terminal/dateUtils.ts` |
