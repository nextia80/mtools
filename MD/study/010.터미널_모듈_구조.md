# 010. 터미널 모듈 구조

명령어가 늘면 `terminalCommands.ts` 한 파일은 비대해집니다. mtools는 **`src/terminal/`** 폴더로 분리합니다.

> 상세 일지: `MD/2026-06-16/2026-06-16_17-00_터미널_명령어_모듈_리팩토링.md`, `2026-06-16_18-00_터미널_소스_구조_상세_설명.md`

---

## 목표

- 명령별 파일 분리 (`bd`, `gc`, `md`, `gm`)
- 공통 HTTP·날짜·도움말 유틸 재사용
- `TerminalView.vue`는 UI만 담당

---

## 폴더 구조

```text
mtools-ui/src/
├── components/TerminalView.vue   # 입력·출력·히스토리 UI
└── terminal/
    ├── index.ts                  # executeTerminalCommand (진입점)
    ├── types.ts                  # TerminalCommandResult, TerminalAction
    ├── context.ts                # cd gc 등 컨텍스트(모드)
    ├── help.ts                   # /? 도움말
    ├── apiClient.ts              # fetchJson, requestJson
    ├── dateUtils.ts              # YYYYMMDD, 기간 계산
    ├── textUtils.ts              # 포맷·검색
    └── commands/
        ├── boardCommand.ts       # bd
        ├── calendarCommand.ts    # gc
        ├── mailCommand.ts        # gm
        ├── mdCommand.ts          # md
        └── loginCommand.ts       # @@login
```

---

## 데이터 흐름

```text
TerminalView.vue
  ↓
terminal/index.ts          ← 명령 분기(라우터)
  ↓
commands/*.ts              ← 도메인별 처리
  ↓
apiClient.ts               ← HTTP
  ↓
http://localhost:8080
  ↓
TerminalLineDraft[]
  ↓
TerminalView.vue           ← 화면 출력
```

---

## index.ts 역할

```typescript
export const executeTerminalCommand = async (
  input: string,
  apiBaseUrl: string,
  context: TerminalContext | null,
  isLoggedIn: boolean,
  session: TerminalSessionInfo,
): Promise<TerminalCommandResult>
```

- 빈 입력, 로그인 필요 여부, `cls`, `cd`, `/?` 등 **공통 처리**
- `bd` → `executeBdCommand`, `gc` → `executeGcCommand` … 위임

---

## apiClient.ts

REST 호출을 한곳에 모읍니다.

```typescript
export async function fetchJson<T>(url: string): Promise<T> {
  const response = await fetch(url)
  if (!response.ok) throw new Error(`HTTP ${response.status}`)
  return response.json() as Promise<T>
}
```

제네릭 `<BoardPost[]>`로 응답 타입을 명시 → TypeScript가 오타를 잡아줍니다.

## cd 컨텍스트 (모드)

```text
> cd gc
(gc)> week          ← gc -s week 와 동일
(gc)> a -t "회의"   ← gc a -t "회의"
> cd                ← 컨텍스트 해제
```

`context.ts`에서 현재 모드(`gc`, `bd` …)를 기억하고, 입력 앞에 접두어를 붙입니다.

## 리팩토링 순서 (학습용)

```text
1. terminalCommands.ts에 명령 추가 (동작 확인)

```
