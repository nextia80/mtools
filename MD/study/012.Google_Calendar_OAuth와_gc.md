# 012. Google Calendar OAuth와 gc

일정을 Google Calendar에서 가져오려면 **OAuth 2.0**으로 사용자 Google 계정 연동이 필요합니다. 터미널에서는 `gc` 명령으로 조회·추가합니다.

> 상세 일지: `MD/2026-06-14/2026-06-14_22-50_일정관리_Google_Calendar_OAuth_연동.md`, `2026-06-22/2026-06-22_18-00_gc_명령어_스펙_재정의_정리.md`

---

## 목표

- Google Cloud Console OAuth 클라이언트 설정
- Spring Boot: auth-url → callback → 토큰 DB 저장
- `gc` 명령으로 일정 조회

---

## 1. OAuth 흐름

```text
[일정관리] Google Calendar 연결
  ↓
GET /api/calendar/auth-url  → Google 로그인 URL
  ↓
사용자 승인
  ↓
GET /api/calendar/oauth/callback?code=...
  ↓
code → access/refresh token → DB (t_google_oauth)
  ↓
프론트 ?view=schedule&calendar=connected
  ↓
GET /api/calendar/events → 일정 표시
```

---

## 2. Google Cloud Console (1회)

1. **Google Calendar API** 활성화
2. **OAuth 동의 화면** — 테스트 사용자에 본인 Gmail 추가
3. **OAuth 클라이언트 ID** (웹 애플리케이션)
4. **리디렉션 URI**:

```text
http://localhost:8080/api/calendar/oauth/callback
```

5. Client ID / Secret → **Git에 올리지 않음**

---

## 3. application-local.properties

```properties
app.google.client-id=YOUR_CLIENT_ID
app.google.client-secret=YOUR_CLIENT_SECRET
```

`application.properties`에서 optional import:

```properties
spring.config.import=optional:classpath:application-local.properties
```

OAuth **없이** 게시판·MD·Echo는 사용 가능합니다.

---

## 4. gc 명령 기본

| 명령 | 동작 |
|------|------|
| `gc` | **오늘** 일정 |
| `gc -l c` | 캘린더(카테고리) 목록 |
| `gc -s 2026.06` | 2026년 6월 |
| `gc -s week` | 오늘 ~ 7일 |
| `gc -s today -c 0` | 오늘 + 카테고리 0번 |
| `/? gc` | gc 도움말 |

일자 표기: **점 구분** (`2026.06.22`, `2026.01~2026.06`)

`cd gc` 후에는 `gc` 접두어 생략 가능 (`week` → `gc -s week`).

---

## 5. 일정 추가 (요약)

```text
gc a -t "회의" -s 2026.06.25 -c 0
```

백엔드 `POST /api/calendar/events` — Google Calendar API에 이벤트 생성.

---

## 6. 확인

1. `application-local.properties` 설정
2. API 재시작
3. 일정관리 메뉴 → Google 연결
4. 터미널: `gc` 또는 `gc -s week`

```bash
curl -s http://localhost:8080/api/calendar/status
```

`connected: true`면 토큰 저장 성공.

---

## 7. 막혔을 때

| 증상 | 확인 |
|------|------|
| redirect_uri_mismatch | Console URI와 properties 일치 |
| 403 Testing | 테스트 사용자에 Gmail 등록 |
| gc 빈 목록 | OAuth 연결, `-s` 날짜 범위 |
| Secret 노출 | MD/Git 커밋 제외, 토큰 재발급 |
